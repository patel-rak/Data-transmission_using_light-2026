## Overview
A wireless data communication system utilizing Light Fidelity (Li-Fi) technology. The system processes text data via an Arduino, transmits it wirelessly over a visible light spectrum using a light-emitting mechanism, and decodes it on the receiver end using an LDR sensor to display the output directly onto an LCD screen.

## Repository Contents
* `data_transmit.ino` - Arduino source code for data encoding and light modulation.
* `circuit_diagram.png` - Schematic blueprint of the hardware connections.

* `Final_report.pdf` - Complete project documentation and theoretical analysis.
